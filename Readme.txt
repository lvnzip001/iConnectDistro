# iPhone Distribution Personal Store

Welcome to the  Distribution Personal Store! This project is a web-based store for distributing iPhones. It is built using CSS, HTML, JS, and the Bootstrap framework.

## Features

- Browse and purchase at competitive prices.
- Secure payment processing for a seamless shopping experience.
- User-friendly interface for easy navigation and product selection.
- Responsive design that adapts to different screen sizes for optimal viewing on various devices.

## Technologies Used

- CSS: Used for styling the web pages and creating an appealing user interface.
- HTML: Provides the structure and content of the web pages.
- JavaScript (JS): Used for interactive functionality and dynamic elements.
- Bootstrap: Utilized as a front-end framework for responsive design and pre-built UI components.
